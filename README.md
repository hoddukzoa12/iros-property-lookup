# IROS 부동산고유번호 조회 서비스

주소를 대량으로 입력하면 관련 **부동산고유번호(14자리)**를 일괄 조회하고, 인터넷등기소 **일괄열람 등록양식(.xls)**을 30건 단위로 자동 생성하는 웹 서비스.

세무·법무 실무에서 주소를 하나씩 검색하던 반복 작업을 없애는 것이 목표입니다. 조회·양식 생성까지만 자동화하며, 이후 인터넷등기소 로그인·결제·열람·저장은 사용자가 직접 수행합니다.

## 주요 기능

- 주소 대량 입력 (붙여넣기 / CSV·XLSX 업로드)
- 주소별 부동산고유번호 전체 수집 (페이지네이션 순회, 중복 제거)
- 결과 검토 — 유형(토지/건물/집합건물) 필터, 집합건물 호실 필터, 제외 토글
- 내보내기 — 전체 데이터 CSV/XLSX, 일괄열람 등록양식 .xls(30건 단위 분할)

## 기술 스택

- **프론트엔드**: React + Vite (Cloudflare Pages)
- **API**: Cloudflare Worker (조회 프록시)
- **엑셀 생성**: SheetJS (클라이언트 사이드, BIFF8 .xls)

## 아키텍처

```
Cloudflare Pages (React SPA)  ──►  Cloudflare Worker (프록시·수집)  ──►  인터넷등기소 검색
        입력·검토·내보내기 UI              페이지네이션·중복제거·정규화
```

브라우저에서 등기소를 직접 호출할 수 없어(CORS) Worker를 프록시로 둡니다. 조회는 Worker, 엑셀 생성은 클라이언트에서 처리합니다.

## 개발

```bash
npm install
npm run dev          # 프론트 + Worker 로컬 실행
```
